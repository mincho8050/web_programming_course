- Window -> Preferences -> General -> Editors -> Text Editors -> Spelling - Encoding 속성을 UTF-8로 변경


- Window -> Preferences -> Web 
                           
	?     -> CSS Files
	?    -> HTML Files
	?    -> JSP Files Encoding을 UTF-8로 변경
