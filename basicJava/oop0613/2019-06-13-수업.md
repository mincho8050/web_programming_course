### dos명령어

- d: > d:드라이브 변경
- dir >  목록
- cd > 폴더변경
- cd\ > root로 폴더 변경
- type > 텍스트파일 내용 출력
- copy > 파일복사 
- del > 파일 삭제
- cls > 화면 clear







###### 1. 파일 내용 콘솔창 출력

java Typing sungjuk.txt

​					 >args[0]

###### 2. 파일 복사

java Copying 	sungjuk.txt	sungjuk2.txt

​		>클래스명	>args[0]	>args[1]

​							>읽기모드	>쓰기모드

###### 3. 파일 삭제

java DeleteFile	sungjuk3.txt

​		>클래스명	>args[0]
